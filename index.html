Étape 1 — Décomposition

Comment décomposer un filtre d'ordre 3 ?

Un filtre d'ordre 3 ne peut pas être réalisé directement par un seul Sallen-Key (ordre 2 max). Il faut le décomposer en cascade. Les fonctions de transfert se multiplient grâce à l'impédance de sortie nulle de l'AOP.

Propriété clé H_total(jω) = H₁(jω) × H₂(jω)
Z_out(AOP) ≈ 0 → aucune interaction entre étages

Étape 2 — Étage 1er ordre

Calculer R pour l'étage passe-bas 1er ordre

L'étage 1er ordre est un filtre RC passe-bas suivi d'un suiveur (gain = 1). Sa fréquence de coupure doit être f₀ = 1 kHz.

Relations clés H₁(jω) = 1 / (1 + jω/ω₀)
ω₀ = 2π f₀ = 2π × 1000 ≈ 6 283 rad/s
ω₀ = 1/(RC) → R = 1/(ω₀·C)
Astuce : on fixe C = 100 nF (valeur commerciale courante), puis on calcule R. C'est la méthode standard en Sallen-Key.
Ω
Étape 3 — Étage Sallen-Key 2ème ordre

Trouver Q et k pour le Butterworth ordre 3

Pour un Butterworth d'ordre 3, l'étage 2ème ordre a un facteur de qualité Q = 1 (différent du Butterworth ordre 2 qui a Q = 0.707 !). Déduisez k à partir de Q.

Sallen-Key passe-bas H₂(jω) = k / [(jω/ω₀)² + jω(3−k)/ω₀ + 1]

Q = 1/(3−k) → k = 3 − 1/Q
ω₀ même f₀ = 1 kHz → même R et C qu'à l'étape 2
